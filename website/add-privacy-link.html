Add Privacy Link to Your Website Footer

Current Footer: "Icumbi © 2025 - "Ubukode bworoshye, ubuzima bwiza.""

Target Footer: "Icumbi © 2025 - "Ubukode bworoshye, ubuzima bwiza."" + Privacy link

Option 1: Simple HTML Addition

Find your current footer HTML and add this after the existing text:

<div style="margin-top: 10px;"> <a href="/privacy" style="color: #007bff; text-decoration: none; margin: 0 15px; font-size: 14px; font-weight: 500;">Privacy</a> <a href="/terms" style="color: #007bff; text-decoration: none; margin: 0 15px; font-size: 14px; font-weight: 500;">Terms</a> <a href="/contact" style="color: #007bff; text-decoration: none; margin: 0 15px; font-size: 14px; font-weight: 500;">Contact</a> </div>

Option 2: Complete Footer Replacement

Replace your entire footer with this:

<footer style="background-color: #f8f9fa; padding: 20px 0; text-align: center; border-top: 1px solid #dee2e6;"> <div style="max-width: 1200px; margin: 0 auto; padding: 0 20px;"> <div style="color: #666; font-size: 16px; margin-bottom: 15px;"> Icumbi © 2025 - "Ubukode bworoshye, ubuzima bwiza." </div> <div style="margin-top: 15px;"> <a href="/privacy" style="color: #007bff; text-decoration: none; margin: 0 15px; font-size: 14px; font-weight: 500;">Privacy</a> <a href="/terms" style="color: #007bff; text-decoration: none; margin: 0 15px; font-size: 14px; font-weight: 500;">Terms</a> <a href="/contact" style="color: #007bff; text-decoration: none; margin: 0 15px; font-size: 14px; font-weight: 500;">Contact</a> <a href="/support" style="color: #007bff; text-decoration: none; margin: 0 15px; font-size: 14px; font-weight: 500;">Support</a> </div> </div> </footer>

Option 3: JavaScript Auto-Addition

Add this script to your website's <head> section:

<script> document.addEventListener('DOMContentLoaded', function() { // Find footer elements const footerElements = document.querySelectorAll('footer, .footer, [class*="footer"]'); footerElements.forEach(footer => { // Look for the existing footer text const textNodes = Array.from(footer.childNodes).filter(node => node.nodeType === Node.TEXT_NODE && node.textContent.includes('Ubukode bworoshye') ); if (textNodes.length > 0) { const textNode = textNodes[0]; const privacyLink = document.createElement('a'); privacyLink.href = '/privacy'; privacyLink.textContent = ' Privacy'; privacyLink.style.cssText = 'color: #007bff; text-decoration: none; font-weight: 500; margin-left: 15px;'; // Insert the link after the text textNode.parentNode.insertBefore(privacyLink, textNode.nextSibling); } }); }); </script>
